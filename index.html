Notes for Block and Inline


All elements are naturally blocks. When you change them to inline, elements will share the same line. When it come to padding though, inline will only create padding on the left and right. If you want padding all around like in a block element, change it to inline-block. When an element is in inline-block, the padding will be applied all around, and the elements will still stay inline.
